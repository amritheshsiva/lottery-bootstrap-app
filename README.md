# Lottery Winner Selector

A simple and responsive Lottery Winner Selector built using **HTML, Bootstrap 4, and JavaScript**.

## Features

- Random lottery winner selection
- Responsive centered layout
- Bootstrap 4 card design
- Modal popup to display winner
- Clean and simple UI

## Technologies Used

- HTML5
- Bootstrap 4
- JavaScript
